# CS CAR SERVICE APP
This example project demonstrates  fleet management using Spring Boot web application

# Pre-requisites
1. Java 1.8 installed
2. Maven version >=3.3.3
3. Spring  : https://start.spring.io/  
4. Eclipse IDE  

# Requirement :
	Need to build Rest Api for fleet management
